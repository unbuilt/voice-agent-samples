
# Hosted Agent Sample - Github Copilot Responses

This sample demonstrates a hosted agent with Responses protocol, integrating with Github Copilot SDK, for a voice live use case.

## Development 

### Init project

```bash
azd ai agent init -m https://github.com/unbuilt/voice-agent-samples/blob/main/samples/github-copilot-python-responses-voicelive/agent.manifest.yaml
```

### Deploy

```bash
azd provision
```

```bash
azd deploy
```
